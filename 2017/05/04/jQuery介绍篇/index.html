jQuery介绍

jQuery是目前使用最广泛的javascript函数库。据统计，全世界排名前100万的网站，有46%使用jQuery，远远超过其他库。从jQuery的口号中我们就可以看出端倪（jquery的口号和愿望 Write Less, Do More（写得少，做得多））。

jQuery的版本分为1.x系列和2.x、3.x系列，目前使用最多的是1.x系列的；1.x系列兼容低版本的浏览器，2.x、3.x系列放弃支持低版本浏览器。

1、http://jquery.com/ 官方网站
2、https://code.jquery.com/ 版本下载

jquery是一个函数库，一个js文件，页面用script标签引入这个js文件就可以使用。
